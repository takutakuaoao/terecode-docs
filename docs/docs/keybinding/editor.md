# Editor Keybinding

## Cursor

| Style | Key |
| :---: | :---: |
| Down | `Ctrl` + `j` |
| Up | `Ctrl` + `k` |
| Left | `Ctrl` + `h` |
| Right | `Ctrl` + `l` |
| Head Line | `Ctrl` + `g` |
| End Line | `Ctrl` + `'` |
| Down Select | `Ctrl` + `Shift` + `j` |
| Up Select | `Ctrl` + `Shift` + `k` |
| Left Select | `Ctrl` + `Shift` + `h` |
| Right Select | `Ctrl` + `Shift` + `l` |
| Head Line Select | `Ctrl` + `Shift` + `g` |

## Block

| Style | Key |
| :---: | :---: |
| Code Block | `Ctrl` + `p` |
| Table Block | `Ctrl` + `t` |

## Other

| Style | Key |
| :---: | :---: |
| Toggle Preview | `Ctrl` + `Shift` + `p` |
